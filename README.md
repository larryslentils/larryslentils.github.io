# Snipcart v3 demonstration store

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## NOTE
May have to use:

```
 NODE_OPTIONS=--openssl-legacy-provider npm run dev
```
